== Description ==

Layouts for Divi is an divi add-on to give you top-niche ready-made landing pages or layouts for your favorite WordPress Divi page builder. This is an simple and flexible way to import in your WordPress site via one click - 'Import Template' or 'Create new Page'.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==
= Can I import this ready made layouts to my Divi Library? =
Yes, you can. Layouts for divi plugin gives you the ability to import any layout to Divi library and pages too!

=  Can I change color or customize imported layouts as per my wish? =
Yes, of course. Why not! You are free to use/modify it as per your project requirements.
